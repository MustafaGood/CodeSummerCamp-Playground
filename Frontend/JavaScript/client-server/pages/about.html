Om Code Summer Camp

Code Summer Camp är ett svenskt sommarläger som syftar till att väcka intresse och nyfikenhet för teknik, programmering, digitalt skapande, kreativitet och problemlösning hos barn och unga, oavsett förkunskaper. Sedan starten har över 1 900 deltagare medverkat i lägret.

Om Code Summer Camp 2025

Lägret erbjuder olika format för att passa olika behov:

Fysiskt läger

Hålls på Linköpings Universitet under veckorna 25–27, med olika åldersgrupper varje vecka.

Digitalt läger

Deltagare födda 2008–2015 kan delta på distans under vecka 25.

Hybridläger

Deltagare sitter på angivna platser i olika städer och är uppkopplade mot den digitala veckan.

Deltagarna får prova på olika programmeringsspråk och delta i inspirerande möten med partners inom industri- och tech/IT-branschen.
